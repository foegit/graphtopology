Опис

Програма призначена для отримання характеристик деякої географічної структури, що можна представити у вигляді графа. Програма рахує наступні топологічні характеристики:

  • Ступінь вершин
  • Індекс абсолютної доступності
  • Число Кеніга
  • Індекс Бавелаша
  • Індекс Бічема
  • Міру оцінки положення регіону в системі
  • Альфа-індекс
  • Бета-індекс
  • Фіта-індекс

Технічна складова

Програма написана на сучасній мові програмування Python (3.7). Також використовуються сучасні наукові бібліотеки для роботи з даними та графами:

  • NumPy
  • Pandas
  • NetworkX

Бібліотека NetworkX швидко та правильно проводить всі необхідні операції з графами. Для пошуку відстаней використовується алгоритм Флойда-Варшелла, а саме його бібліотечна реалізація networkx floyd_warshall_numpy().

Бібліотека NumPy слугує для зберігання матриці суміжності та надає зручні функції для робити з масивами даних такі як sum() та max(). Використання бібліотеки також дозволяє користуватись низькорівневими конструкціями, що в перспективі вплине на швидкодію програми.

Бібліотека Pandas забезпечує коректне та правильне зчитування Excel-файлу. Також бібліотека зручно взаємодіє з numpy.
